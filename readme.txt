
Выбираем ник юзера, у которого хотим найти все комменты



Программа:
    1. Определяет количество комментариев юзера.
    2. Находит номер самого свежего поста
    3. Просматривает посты, записывает найденные урлы комментариев юзера в
    файл

Выход из программы:
    1. Найти все комментарии
    2. Просмотреть заданное количество постов и вывести найденные
    комментарии
    3. Просмотреть посты на заданном промежутке времени
